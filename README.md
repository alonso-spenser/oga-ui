# Guide

----

`OGA UI` is a frontend UI component library built on `Vue 3` and `Element Plus`.


## Install

```bash
npm i oga-ui
```

## How to use

in `main.ts`

```js
import "oga-ui/index.css"
import ogaUI from 'oga-ui'

app.use(ogaUI)
```

## What's new

Input
